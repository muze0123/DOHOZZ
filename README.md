一、产品名称
DOHOZZ

二、Slogan
精准达人营销，智能决策驱动社媒电商增长

三、产品介绍
DOHOZZ 是一款专为品牌商家设计的达人营销智能决策与管理平台，支持 TikTok、Shopee、Instagram、Lazada等多平台的达人营销管理。作为一站式解决方案，DOHOZZ 助力用户高效筛选海量达人，精细化管理合作流程，并实时追踪分析营销效果，推动品牌内容与电商业务的双重增长。

四、解决问题
数据统计难题：跨平台店铺数据难以整合，DOHOZZ 提供统一的数据分析界面，帮助您轻松获取各平台的营销效果。
业务方向明确：通过智能决策工具，您可以快速洞察市场趋势，确保品牌始终走在市场前沿。
达人筛选高效：利用智能算法，DOHOZZ 可以快速而准确地为您推荐最适合的达人，节省时间和精力。
直播内容跟踪：实时记录直播中的关键内容，帮助您更好地了解主播的表现和受众反应。
减少错误，提高效率：通过自动化管理工具，减少手动表格跟进带来的错误，提升工作效率。

五、核心价值
通过智能化、自动化的工具和数据驱动的管理方式，帮助企业高效管理达人资源，优化合作流程，提升品牌出海和营销转化效率。
1. 高效的达人资源管理
智能筛选与匹配：通过AI算法从海量达人库中筛选出符合品牌需求的优质达人，基于粉丝画像、互动数据、内容风格等多维度实现精准匹配。
全流程管理：覆盖从达人线索挖掘、建联、寄样、履约到合作单生成的全流程，提供一站式管理体验，提升工作效率。
2. 数据驱动的决策支持
实时数据监控：平台实时跟踪达人合作的各项指标，包括视频发布情况、播放量、互动率、销售转化率等，帮助企业动态评估合作效果。
数据分析与业绩归属：通过数据分析功能，清晰展现团队和个人的业绩表现，支持企业进行科学的资源分配和绩效管理。
3. 智能化运营支持
AI辅助运营：利用AI Agent技术，自动化完成达人邀约、寄样跟进、视频审核、合作单管理等重复性工作，减少人工干预，提高运营效率。
智能提醒与优化建议：提供智能提醒，帮助BD及时处理待办任务，同时根据数据表现给出优化建议，提升运营效果。
4. 提升品牌出海竞争力
全球达人网络：平台整合全球优质达人资源，特别是新兴市场（如东南亚、欧美）的本地化达人，助力中国品牌快速进入国际市场。
供应链与营销整合：通过对接供应链、品牌管理和社媒营销，帮助企业实现从产品到消费者的全链路闭环。
5. 团队协作与透明化管理
多角色权限管理：根据不同角色（如超级管理员、团队负责人、BD等）分配不同的权限和数据可视范围，保障团队协作的高效性和数据安全。
任务分配与进度跟踪：支持团队负责人对BD的任务分配与实时跟踪，确保团队工作有序开展。
6. 提升企业效益
降低人工成本：通过智能化运营工具，减少重复性工作，降低人力成本。
提升转化率：精准匹配优质达人，优化合作流程，提升带货视频转化率和销售额。
缩短合作周期：通过高效的寄样、履约和业绩归属管理，加快合作进程，提升企业整体运营效率。

六、用户画像
1. 主要用户群体
DOHOZZ平台的目标用户主要是从事电商、品牌推广和跨境电商业务的企业及其相关从业者。具体包括：
（1）品牌商/电商企业
特点：自有产品或品牌，目标是通过达人营销提升市场影响力和销售额。
需求：
快速找到与品牌定位和目标市场匹配的优质达人。
管理从达人筛选到合作履约的全流程，提升运营效率。
监控达人带货效果，优化营销策略。
行业分布：
美妆、个护、时尚服饰、母婴、数码、家居等领域。
规模：
从中小型品牌到大型跨国企业，尤其是有出海需求的品牌商。
（2）MCN机构/代理公司
特点：专注于达人孵化和运营，帮助品牌对接达人资源，策划并执行营销活动。
需求：
管理旗下达人资源，提升达人合作效率。
提供数据分析支持，优化达人内容表现和商业转化。
批量化处理达人邀约、寄样及合作单管理，降低运营成本。
（3）跨境电商从业者
特点：专注于跨境市场，利用达人营销快速打开海外市场。
需求：
快速找到符合目标市场（如东南亚、欧美等）的本地化达人。
监控达人发布的带货内容，分析其在目标市场的表现。
通过数据分析优化产品定价和营销策略，提升海外市场的竞争力。
2. 角色画像
（1）超级管理员
特点：
企业高层管理者，如CEO、CMO等。
关注全局数据、团队表现和业务发展方向。
行为：
查看全企业的达人资源和业绩数据。
配置系统权限，分配角色。
监控整体运营效率，调整策略。
（2）管理员
特点：
负责企业内部的达人资源管理和团队支持。
行为：
管理达人库、分配线索。
监控团队任务完成情况。
处理合作单审批、达人资源分配等事务。
（3）团队负责人
特点：
负责具体业务部门的管理工作，带领团队完成业绩目标。
行为：
查看本部门及下属BD的达人管理数据。
分配和追踪团队任务。
监控团队成员的业绩和任务完成情况。
（4）BD负责人
特点：
负责达人合作的整体推进，直接对接达人的主要负责人。
行为：
筛选达人资源，制定合作策略。
跟进达人邀约、寄样、履约和合作单管理。
监控达人表现，优化合作模式。
（5）BD（商务拓展人员）
特点：
一线执行人员，负责具体的达人沟通和合作推进。
行为：
跟进分配到的达人线索。
负责与达人建联、寄样、合作和履约的具体操作。
记录达人合作进度，提交合作单。
3. 用户痛点
达人筛选困难：海量达人资源难以快速找到符合品牌需求的对象。
合作流程繁琐：达人邀约、寄样、履约等环节复杂，人工操作耗时长，效率低。
数据管理分散：达人合作数据、业绩归属数据分散在多个系统，难以统一管理和分析。
跨部门协作不畅：团队间任务分配和数据共享不清晰，导致重复劳动和资源浪费。
4. 用户需求
提供精准的达人筛选和匹配功能，快速找到目标达人。
通过自动化工具简化达人邀约、寄样、履约等流程，提升效率。
提供实时数据监控和分析工具，帮助优化达人合作效果。
支持多角色权限管理，确保数据安全和团队协作顺畅。
提供清晰的任务提醒和跟进功能，帮助BD高效完成每日工作。
