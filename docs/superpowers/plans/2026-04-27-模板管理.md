# 模板管理 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the 模板管理 page with 3 tabs (触达/邀约/邮件模板), search, table with 9 columns, pagination, CRUD dialogs, and integrate into Dashboard sidebar.

**Architecture:** Single `TemplateManagement.vue` page with 3 sub-tabs. Dialogs as inline `<el-dialog>` within the same file. Follow existing DOHOZZ patterns (Composition API, Element Plus, scoped SCSS).

**Tech Stack:** Vue 3 `<script setup>`, Element Plus, SCSS, Vue Router
