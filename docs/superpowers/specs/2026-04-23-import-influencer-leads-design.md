# 导入达人线索 - 设计规格

**版本**: v1.0
**创建日期**: 2026-04-23
**状态**: 待审核

---

## 1. 功能概述

### 1.1 背景
导入达人线索功能支持批量导入达人数据，管理达人线索，支持添加达人线索、批量导入达人、批量分配、批量放入公海、批量删除等操作。

### 1.2 目标
- 展示全平台达人线索数据，支持以平台区分达人线索数据
- 支持单个添加达人线索
- 支持批量导入达人线索
- 支持多维度筛选和查询
- 支持批量操作（分配、放入公海、删除）
- 展示达人详细信息及带货商品数据

### 1.3 页面入口
- 导航路径：达人合作 > 找达人 > 导入达人线索
- 入口方式：侧边栏二级菜单「导入达人线索」

## 7. 边界条件处理

| 场景 | 处理方式 |
|------|---------|
| 筛选后无数据 | 达人列表显示空状态，居中显示「暂无数据」，颜色#999999，图标64x64px，筛选项保留 |
| 数据加载中 | 列表区显示骨架屏，灰色背景，高度64px，带有动画效果 |
| 网络异常 | Toast提示「网络异常，请稍后重试」，颜色#FF0000，持续时间3秒 |
| 批量操作未选中达人 | 「批量操作」按钮置灰禁用，背景色#F5F5F5，颜色#CCCCCC |
| 批量分配未选择BD | Toast提示「请选择分配BD」，弹窗保持打开状态 |
| 达人数据为0 | 显示「0」，颜色#333333 |
| 搜索框输入达人ID | 精准搜索，完全匹配达人ID |
| 搜索框输入达人名称 | 模糊搜索，包含关键词即可匹配 |
| 导入达人线索重复导入 | 校验达人ID是否已存在，若已存在则提示「达人已存在，跳过该达人」 |
