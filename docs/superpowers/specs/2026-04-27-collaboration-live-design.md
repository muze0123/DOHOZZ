# 合作直播页面设计规格

## 1. 概述

### 1.1 页面定位
- **页面名称**：合作直播（Collaboration Live）
- **所属模块**：内容中心 / 合作内容 / 合作直播
- **支持平台**：TikTok（默认）/ Instagram / Shopee / Lazada

### 1.2 核心功能
管理品牌合作达人的带货直播，追踪直播成交数据、投放数据、观播数据及归属信息。

### 1.3 与其他合作内容的差异
| 维度 | 带货视频 | 非挂车视频 | 合作直播 |
|------|---------|-----------|---------|
| 内容类型 | 达人带货短视频 | 品牌种草/测评短视频 | 达人带货直播间 |
| 数据来源 | 系统自动监控 + 手动添加 | 仅手动添加 | 系统自动监控 |
| 特有指标 | 销量/退款率/佣金 | 合作费用/小蓝词 | 直播时长/观看人次/UV价值/直播GMV占比/录屏 |

## 3. 区域详细规格

### 3.1 区域A — 平台Tab切换栏

| 属性 | 说明 |
|------|------|
| Tab项 | TikTok（默认激活）/ Instagram / Shopee / Lazada |
| 样式 | 平台Logo图标 + 平台名称，激活Tab蓝色下划线 + 加粗 |
| 切换行为 | 激活Tab切换；所有筛选条件完全重置为默认值；数据概览区与列表区重新加载 |

### 3.2 区域B — 全局筛选工具栏

**11个筛选控件**：

| 控件名称 | 控件类型 | 说明 | 是否本页特有 |
|---------|---------|------|------------|
| 店铺名称 | 下拉单选 | 筛选指定店铺下的直播数据 | 否 |
| 达人搜索 | 输入框 + 搜索 | placeholder：「请输入达人名称/达人ID」 | 否 |
| 所属BD | 下拉单选 | 筛选指定BD员工归属的直播 | 否 |
| 所属部门 | 下拉单选 | 筛选指定部门归属的直播 | 否 |
| 商品搜索 | 输入框 + 搜索 | placeholder：「请输入商品名称/商品ID」 | 否 |
| 销量 | 下拉面板（快捷+自定义） | 不限 / 0-10 / 10-100 / 1000以上 / 自定义 | 否 |
| 成交金额 | 下拉面板（快捷+自定义） | ¥0-500 / ¥500-2000 / ¥2000-5000 / ¥5000以上 / 自定义 | 是 |
| 是否有投放 | 下拉单选 | 全部 / 是 / 否 | 否 |
| 发布时间 | 日期区间选择器 | 格式：YYYY/MM/DD-YYYY/MM/DD | 否 |
| 是否有寄样 | 下拉单选 | 全部 / 是 / 否 | 是 |
| 有录屏 | 开关 | 筛选有直播录屏文件的直播场次 | 是 |

**交互规则**：
- 选择店铺/BD/部门：立即触发全页数据刷新
- 达人搜索/商品搜索：回车触发搜索，清空后恢复全量数据
- 销量/成交金额自定义：最小值不可大于最大值
- 切换平台Tab：所有筛选条件完全重置为默认值

### 3.3 区域C — 追踪统计横幅

- 文案格式：「系统已为您累计追踪到 [X] 位达人的 [Y] 个视频及 [Z] 场直播」
- 数字以蓝色加粗大字展示
- 右侧快捷入口：【我的追踪达人】【监控规则】

### 3.4 区域D — 数据概览区

**统计范围**：
- 文案：「统计在 YYYY/MM/DD - YYYY/MM/DD 期间开播的达人直播间数据」
- 日期范围随全局筛选中「发布时间」条件动态变化
- **默认时间范围：近30天**

#### 合作直播数据汇总（6个指标）
| 指标名称 | 展示格式 |
|---------|---------|
| 直播场次 | 整数，千分位分隔 |
| 直播成交金额 | ¥1,299.52 |
| 直播销量 | 整数 |
| 直播退款金额 | ¥310.76 |
| 预估达人佣金 | ¥458.66 |
| 直播达人数 | 整数 |

#### 投放直播数据汇总（5个指标）
| 指标名称 | 说明 |
|---------|------|
| 投放直播数 | 所选时间内有投放的直播场次数量 |
| 消耗金额 | 所选时间内所有投放直播的累计投流消耗金额 |
| 成交金额（投放） | 所选时间内所有投放直播累计投流成交金额 |
| 成交订单数 | 所选时间内所有投放直播累计投流成交订单数 |
| 支付ROI | 总投放成交金额 / 总消耗金额 |

### 3.5 区域E — 直播列表区

#### 列表子Tab
| Tab | 数量角标 | 数据范围 |
|-----|---------|---------|
| 全部（n） | 当前筛选下的直播总场次 | 展示所有合作直播 |
| 有出单（n） | 有成交记录的直播场次数 | 展示所选时间内有合作商品成交的直播 |
| 未出单（n） | 无成交记录的直播场次数 | 展示所选时间内合作商品无成交的直播 |

#### 列表顶部工具栏
- 直播标题搜索：输入框，placeholder「请输入直播标题」，回车触发搜索
- 【导出】按钮：导出当前筛选结果（Excel 格式）
- 「是否有投放」筛选：与全局筛选中的「是否有投放」**联动**

#### 列表列字段
| 列名 | 格式/说明 |
|------|---------|
| 视频 \| 直播时间 | 直播封面缩略图 + 直播标题（超长截断省略）+ 直播时间（YYYY-MM-DD HH:mm） |
| 达人 | 圆形头像 + 达人用户名 + 达人名称 + 粉丝数（如「12w」） |
| 合作商品 | 直播中挂链的合作商品主图缩略图；hover弹出商品卡Tooltip |
| 归属员工 | 展示该直播的归属员工姓名；无归属时显示「- -」 |
| 成交金额 + GMV占比 | 主行：¥250；次行小字：直播GMV占比：0.02% |
| 销量 | 整数 |
| 退款金额 + 退款占比 | 主行：¥10；次行小字：退款占比：0% |
| 预估达人佣金 | ¥8 |
| 是否有投流 | 是 / 否 |
| 消耗金额 | ¥0 或 - - |
| 成交金额（投放） | - - |
| 成交订单数（投放） | - - |
| 支付ROI | - - |
| 直播时长 | X小时XX分XX秒（如「3小时28分56秒」） |
| 观看人次 | 以「w」为单位（如「16.15w」） |
| 预估UV价值 | 待定 |
| 操作 | 【详情】（新标签页打开） |

#### 列表交互
- 鼠标悬停列表行：行背景变为浅蓝色高亮
- 鼠标悬停合作商品封图：弹出商品卡Tooltip（移出延迟100ms关闭）
- 鼠标悬停带【?】列头：弹出对应字段说明Tooltip
- 点击达人头像/用户名：跳转至达人详情页（新标签页）

#### 列表分页
- 底部分页信息：「共 N 条记录 第 X/Y 页」
- 每页条数下拉：10 / 20 / 50，默认10条/页
- 页码导航：上一页 / 页码列表（最多5个，超出显示「...」+ 末页）/ 下一页
- 快速跳页：输入框「跳至 [n] 页」+ 回车执行

## 4. Toast提示规范

| 触发场景 | Toast文案 | 类型 |
|---------|----------|------|
| 导出触发成功 | 导出成功，请查收 | 成功（绿） |
| 跳页输入超出范围 | 请输入有效页码 | 警告（橙） |
| 数据加载失败 | 数据加载失败，请刷新重试 | 失败（红） |

---

## 5. 空状态规范

| 场景 | 展示内容 |
|------|---------|
| 列表无数据 | 空状态插图 + 「暂无相关直播」+ 「请调整筛选条件后重试」 |
| 数据概览无数据 | 各指标显示「——」 |
| 搜索标题无结果 | 空状态 + 「未找到匹配结果，请调整筛选条件」 |

---

## 6. 加载状态规范

| 场景 | 展示内容 |
|------|---------|
| 列表加载中 | 骨架屏（Skeleton）代替行数据 |
| 数据概览区加载中 | 各指标区域显示Loading骨架条 |
| 导出文件生成中 | 按钮进入Loading态，完成后自动下载 |

---

## 7. 数据格式规范

| 类型 | 规范 |
|------|------|
| 金额 | 保留2位小数，千分位逗号，前缀¥（如「¥1,299.52」） |
| 销量/场次/达人数 | 整数，千分位分隔（如「1,243」） |
| 占比 | 保留2位小数百分比（如「0.02%） |
| 观看人次/UV价值 | 以「w」为单位（如「16.15w」） |
| 直播时长 | X小时XX分XX秒格式（如「3小时28分56秒」） |
| 无数据字段 | 显示「- -」 |
| 时间 | 「YYYY-MM-DD HH:mm」格式 |
