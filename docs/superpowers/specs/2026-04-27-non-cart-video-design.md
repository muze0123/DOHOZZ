# 非挂车视频页面设计文档

## 1. 概述

### 1.1 页面定位

「非挂车视频」是 DOHOZZ 达人营销平台「内容中心 - 合作内容」模块下的二级菜单页面，专门管理达人发布的无商品链接视频（纯品牌种草、测评等内容形式）。

### 1.2 核心功能

- 平台维度筛选（TikTok / Instagram / Shopee / Lazada）
- 全局时间筛选（近7天/周/月/自定义）
- 分析区指标卡 + 折线趋势图
- 视频列表筛选、排序、分页
- 手动添加非挂车视频链接
- 投流平台同步管理
- 互动数据手动刷新（有频率限制）

### 1.3 与带货视频的核心差异

| 维度 | 带货视频 | 非挂车视频 |
|------|---------|-----------|
| 视频来源 | 自动监控 + 手动添加 | 仅手动添加 |
| 指标块数量 | 3个（新发布/动销/投放） | 2个（全部视频/新发布视频） |
| 成交数据 | 有（成交金额/退款率/佣金/销量） | 无 |
| 合作费用 | 无 | 有（行内编辑） |
| 小蓝词 | 无 | 有（行内编辑） |
| 投放同步列 | 无 | 有（已同步/未同步状态） |
| 互动数据手动刷新 | 有频率限制 | 有（每10分钟/视频） |

NonCartVideo.vue

## 3. 区域详细设计

### 3.1 平台 Tab 切换栏（区域 A）

**展示内容：**
- 横向 Tab：TikTok（默认激活）/ Instagram / Shopee / Lazada
- 每个 Tab 显示平台 Logo 图标 + 平台名称
- 激活 Tab：蓝色下划线高亮 + 文字加粗

**交互规则：**
- 点击其他平台 Tab → 激活态切换，筛选条件完全重置为默认值，分析区与列表区重新加载
- 切换时显示全页 Loading
- 某平台暂无数据：列表区展示空状态，分析区指标显示「——」

### 3.2 全局时间筛选（区域 B）

**控件说明：**
- 选项：周 / 月 / 自定义 / 近N天
- 默认「近7天（YYYY/MM/DD - YYYY/MM/DD）」
- 选中快捷选项时，右侧展示具体日期区间
- 选择「自定义」时弹出日期区间选择器

**交互规则：**
- 选择快捷选项 → 自动计算日期区间并展示，立即触发分析区与列表区数据刷新
- 选择「自定义」→ 弹出日期区间选择器，确认后触发刷新
- 切换平台 Tab → 时间筛选重置为默认值（近7天）

### 3.3 分析区（区域 C）

#### 3.3.1 数据元信息

- 数据更新时间：每天凌晨 1 点更新（格式：「数据更新时间：YYYY-MM-DD HH:mm」）
- 数据范围：「数据范围：YYYY/MM/DD - YYYY/MM/DD」（随全局时间筛选动态变化）

#### 3.3.2 两指标数据块

| 指标名称 | 说明 | 默认选中 |
|---------|------|---------|
| 全部视频 | 公司员工累计添加的非挂车视频数 | 是 |
| 新发布视频 | 视频发布时间在统计时间段内的视频数 | 否 |

**交互规则：**
- 点击某指标块 → 该块进入选中态（蓝色边框高亮），其余块恢复默认
- 同一时间只能选中一个指标
- 默认进入页面 → 「全部视频」为选中态
- 切换平台 Tab → 指标块重置为「全部视频」

#### 3.3.3 全部视频数据 — 汇总子指标

| 子指标 | 说明 |
|--------|------|
| 视频播放量 | 在所选时间内监控到的播放量；单个视频数据更新15天 |
| 视频点赞数 | 在所选时间内监控到的点赞数；单个视频数据更新15天 |
| 视频评论数 | 在所选时间内监控到的评论数；单个视频数据更新15天 |
| 视频收藏数 | 在所选时间内监控到的收藏数；单个视频数据更新15天 |

#### 3.3.4 折线趋势图

- X 轴：日期（按筛选时间粒度展示）
- Y 轴（左）：视频数量（柱状图）
- Y 轴（右）：互动量（折线图）
- 图例：视频数 / 视频播放量 / 视频点赞数 / 视频评论数 / 视频收藏数
- 鼠标悬停某日数据点 → 弹出 Tooltip 展示当日详细数据

### 3.4 列表区（区域 D）

#### 3.4.1 列表标题

随上方指标块联动：
- 「全部视频」选中时 → 「非挂车视频列表-全部视频」
- 「新发布视频」选中时 → 「非挂车视频列表-新发布视频」

## 4. 弹窗设计

### 4.1 添加非挂车视频链接弹窗

**触发方式：** 点击列表区顶部工具栏【添加非挂车视频】按钮

**弹窗字段：**

| 字段 | 是否必填 | 说明 |
|------|---------|------|
| 视频链接 | 必填 | 输入对应平台的视频链接；提交后自动识别视频基本信息 |
| 达人 | 必填或自动识别 | 从视频链接识别达人，或手动选择绑定达人 |
| 合作费用 | 可选 | 输入该视频对应的达人合作报酬金额 |
| 归属员工 | 可选 | 指定该视频的归属员工 |
| 小蓝词 | 可选 | 为该视频添加关键词标签（可多个） |

**交互规则：**
- 输入视频链接后失去焦点 → 自动识别视频基本信息（封图/标题/达人等），回显在弹窗内
- 识别失败时 → 显示「链接无效，请重新输入」，用户可重试，弹窗保持
- 点击【确定】提交 → 校验必填项，通过后录入系统，弹窗关闭，列表刷新，Toast「添加成功」
- 点击【取消】/关闭 → 弹窗关闭，数据不提交

### 4.2 投流数据同步弹窗

**触发方式：** 点击列表区顶部工具栏【投流平台同步】按钮

**弹窗字段：**

| 字段 | 说明 |
|------|------|
| 同步视频列表 | 展示待同步的视频（封图 + 标题 + 发布时间），无数量上限 |
| 投流平台 | 选择同步目标平台（TikTok 等，与当前页面 Tab 对应） |

**交互规则：**
- 点击【确定】同步 → 发起同步请求
- 同步成功后 → 弹窗关闭、列表刷新、对应视频「投放同步」列变更为「已同步」、Toast「同步成功」
- 同步失败时 → Toast「同步失败，请稍后重试」
- 点击【取消】/关闭 → 弹窗关闭，不执行同步

## 5. 交互规则汇总

### 5.1 Toast 提示汇总

| 触发场景 | Toast 文案 | 类型 |
|---------|-----------|------|
| 添加非挂车视频成功 | 添加成功 | 成功（绿） |
| 视频链接识别失败 | 链接无效，请重新输入 | 失败（红） |
| 投流平台同步成功 | 同步成功 | 成功（绿） |
| 投流平台同步失败 | 同步失败，请稍后重试 | 失败（红） |
| 导出触发成功 | 导出成功，请查收 | 成功（绿） |
| 互动数据手动更新成功 | 互动数据已更新 | 成功（绿） |
| 跳页输入超出范围 | 请输入有效页码 | 警告（橙） |

### 5.2 空状态规范

- 列表无数据：空状态插图 + 「暂无相关视频」 + 「请点击【添加非挂车视频】进行添加」
- 分析区无数据：指标显示「——」；趋势图展示空提示「暂无数据」
- 搜索无结果：列表空状态 + 「未找到匹配结果，请调整筛选条件」

### 5.3 加载状态规范

- 列表加载中：骨架屏（Skeleton）代替行数据
- 分析区图表加载中：图表容器内旋转 Loading 或骨架条占位
- 互动数据手动更新中：对应行互动数据列显示旋转 Loading，完成后刷新数值（有频率限制）
- 同步中：弹窗内【确定】按钮进入 Loading 态，不可重复点击

### 5.4 数据格式规范

- 金额：保留 2 位小数，千分位逗号，前缀 ¥（合作费用可以「w」为单位展示，如「¥2w」）
- 互动数量：总量 + 所选时间增量（绿色「+数字」/红色「-数字」）
- 粉丝数：以「w」为单位（如「12w」）
- 时间：绝对时间「YYYY-MM-DD HH:mm」；同步时间精确到秒「YYYY-MM-DD HH:mm:ss」
- 无数据字段：显示「- -」

### 5.5 互动数据手动刷新频率限制

- 每视频每 10 分钟最多触发一次手动刷新
- 刷新中显示旋转 Loading
- 频率超限提示：「操作太频繁，请稍后再试」

## 9. 实现说明

### 9.1 行内编辑实现

合作费用、归属员工、小蓝词三列支持行内编辑：
- 单击单元格 → 进入编辑态（输入框/下拉选择）
- 失去焦点或回车 → 保存
- ESC → 取消编辑
- 编辑状态时行背景高亮

### 9.2 频率限制实现

前端维护 `refreshTimestamps` Map（videoId → timestamp），每次触发前检查：
- 当前时间 - 上次刷新时间 < 10 分钟 → 提示「操作太频繁」
- 达到阈值 → 刷新，更新时间戳

### 9.3 「已同步」Tooltip 实现

使用 Element Plus 的 Tooltip 组件，content 通过 slot 传入，显示同步人（当前用户）和同步时间（服务器返回）。
