# 达人详情-基础分析 页面PRD交互说明

## 文档信息
- **页面名称**: 达人详情-基础分析
- **页面类型**: 数据分析详情页
- **框架**: Vue 3 (Composition API)
- **样式方案**: Tailwind CSS

## 三、达人信息区块

### 3.1 吸顶效果
当页面滚动导致达人信息区块隐藏后：
- 在页面**顶部置顶显示**
- 显示内容：达人头像、达人用户名、操作按钮等
- 高度：70px

### 3.2 信息展示结构
```
┌────────────────────────────────────────────────────────────────────────┐
│ [头像]  达人名称    LV8  │  联系方式图标  │  [跟进达人] [分配达人] [+公海]│
│        达人用户名         │               │  [二维码] [主页]             │
└────────────────────────────────────────────────────────────────────────┘
```

### 3.3 字段说明
| 字段 | 展示内容 | 交互 |
|------|----------|------|
| 达人头像 | 显示头像图片 | 点击新开标签页打开TikTok主页 |
| 达人用户名 | username | 点击新开标签页打开TikTok主页 |
| 达人名称 | name | 无 |
| 达人等级 | LV8 (评分) | 无，有评分才显示 |
| 联系方式 | 各平台图标 | 点击弹窗显示联系方式 |
| 更新时间 | 精确到年月日时分秒 | 点击【刷新】更新数据 |
| 首次发布视频时间 | 精确到年月日时分秒 | 鼠标移入【？】气泡显示说明 |
| 达人简介 | 单行显示 | 点击【展开】全显示 |
| 粉丝数 | 数字 | 无 |
| 获赞数 | 数字 | 无 |
| 近30日成交金额 | 金额 | 无 |
| 近30日销量 | 数量 | 无 |
| 带货类目 | 以「、」隔开 | 无 |
| 带货方式 | 视频带货为主/直播带货为主 | 无 |
| 视频平均播放量 | 数字 | 无 |
| 直播场均观看人数 | 数字 | 无 |

### 3.4 操作按钮
| 按钮 | 状态 | 说明 |
|------|------|------|
| 跟进达人 | 蓝色按钮 | 点击校验是否可跟进 |
| 分配达人 | 边框按钮 | 达人已被跟进则隐藏 |
| +达人公海 | 边框按钮 | 已在公海则置灰「已在公海」 |
| 二维码 | 图标 | 鼠标移入气泡显示二维码 |
| 达人主页 | 图标 | 新开标签页打开TikTok主页 |
| 刷新 | 文字链接 | 更新数据，按钮变更为「数据更新中…」置灰 |

## 八、交互动效

### 8.1 页面滚动交互
1. **达人信息吸顶**: 当区块滚出视口，顶部置顶显示简化版达人信息
2. **左侧菜单吸边**: 当区块滚出视口，左侧菜单悬浮固定

### 8.2 按钮状态
| 状态 | 样式变化 |
|------|----------|
| 默认 | 白底+蓝边框+蓝字 或 蓝底+白字 |
| Hover | 透明度 90% |
| Active | 透明度 80% |
| 禁用 | 灰色背景 + 灰色文字 + cursor: not-allowed |

### 8.3 Toast提示
- 位置: 页面顶部居中
- 背景: #555555
- 文字: #FFFFFF
- 圆角: 4px
- 示例: 「更新成功，请在30分钟后刷新页面查看最新数据」
